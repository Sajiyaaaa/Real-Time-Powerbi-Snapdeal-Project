# Real-Time-Powerbi-Snapdeal-Project

# 📌 Overview

This project is an end-to-end data analytics solution that collects product data from Snapdeal using Python and visualizes insights using Power BI.

The project demonstrates:

- Web scraping using Selenium

- Data processing with Pandas

- Business intelligence dashboard creation in Power BI

# 🛠️ Technologies Used

- Python

- Selenium

- Pandas

- Power BI

- WebDriver Manager

# 📂 Project Files

- snapdeal_products.py – Python script for scraping product data

- snapdeal_products_data.csv – Scraped dataset

- snapdeal dashboard.pbix – Power BI dashboard file

# 📊 Dashboard Insights

- The Power BI dashboard provides:

- Category-wise product analysis

- Price and discount comparison

- Rating and review distribution

- Seller performance overview

- Availability tracking

# 🎯 Learning Outcomes

- Real-time web scraping implementation

- Handling dynamic web pages

- Data transformation and cleaning

- Dashboard design and data visualization
